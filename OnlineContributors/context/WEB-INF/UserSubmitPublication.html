Publication Manager

Here is an overview of the ToL pages you have worked on. Other contributors with editing privileges for a given branch may also have made changes to these pages. Click on a preview link to view the current version of the page on the ToL Development Site (it will open in a new window).

The pages listed inside a grey box constitute a publication batch. By default a publication batch will be published in its entirety when you click on the Publish checked pages button; however, if you remove the check mark from a page listing, it will be withheld from publication. Note that the publication of some pages may be dependent upon the publication of others; i. e., you cannot publish the page for a group if the page for the containing group has not yet been published.

If you see a red message next to a page listing, this means that the node that this page is attached to is currently checked out by a ToL contributor. In order to be able to publish such pages, you need to request that the contributor checks the relevant nodes back in. To check in a downloaded branch, open the file in TreeGrow, and select Check in Tree from the File menu.
