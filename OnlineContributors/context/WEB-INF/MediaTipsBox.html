The information provided in this form will be stored with your in the ToL database. It will be used to generate captions, and people can search the ToL for s meeting certain criteria. Click on the lightbulbs to get more information about each section of this form.

If you are uploading multiple s with similar data, you can Copy Data From Another .

Tips for filling out this form

You are not expected to put information into every single one of the fields below. Different fields are relevant for different s. Just fill in the ones that are important to you. Sections of special importance for ToL project administration are highlighted in red: Copyright Information, Use Permitted by Copyright Owner, Attached to Groupsand Document Content.

You can use the Copy Data link above to copy data from another one of your s to the current . This will make things a lot easier when you upload multiple s of the same specimen, for example. Note that any data you may already have entered in the current form will be overwritten with the copied data, so use the Copy Data tool before entering any data, and be sure to check your data to avoid applying the wrong data to a file!

If you are planning on uploading a lot of images, check out the image batch uploading tools. Batch uploading tools for other media types will soon become available.
