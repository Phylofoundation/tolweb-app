    Upload /Edit Media

    Upload a new media file

    To upload a new media file follow the steps below:

    Images
    Please upload only JPEG, GIF, or PNG files. Ideally your master image should be around 800 pixels wide or high (example). See The Multimedia Reference and Adding Media for more information.
    Documents
    Select this media type for uploading text files, PDF, Word, Excel, PowerPoint documents, etc.
    Size limit
    You may upload media files up to 5 megabytes. Most sound and movie files under one minute will fit this requirement. If you would like to contribute a larger file, contact for instructions.
     

    Edit data for one of my media files in the ToL database

    To edit a media file you must first find it in the ToL database. Type search criteria in one of the search fields below and click find media. We recommend you type search criteria into only one field. If you are having trouble finding media files you have uploaded to the ToL visit your media contributor page to view all of the media files that you have contributed to the ToL.

    Note: You will only be able to edit images that you have contributed yourself or for which you own the copyright.

    Delete one of my media files from the ToL database

    To delete a media file you must first find it in the ToL database. Type search criteria in one of the search fields below and click find media. We recommend you type search criteria into only one field.

    Note: You will only be able to edit images that you have contributed yourself or for which you own the copyright.
