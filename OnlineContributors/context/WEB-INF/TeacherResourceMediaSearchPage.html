Add A Document

  • Find a document in the ToL database
  • Add a new document to the ToL database
  • Use the fields below to find or upload the document you want.
  • Then insert the document into your page by typing TOLMEDIA and the ID# of the document.

Find Documents

Documents displayed on ToL pages are retrieved from the ToL media database. Note that not all ToL documents are available for use on all ToL pages. Your search will retrieve only documents that have been released by their copyright owners for your intended use.

Add a New Document to the ToL Database

You can also add a new document to the ToL database. Once the document is uploaded, come back to this page, and find your document using the file name or document ID#.

  1. Click browse to find and select a file to upload from your computer.
  2. Then click on the Upload New Document button.
