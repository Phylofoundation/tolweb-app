Default Licensing Options for

Please select the preferred licenses for your Tree of Life contributions below and then click on the Submit Preferences button at the bottom of this page. These licenses specify how we can use and share your contributions. The licenses you select here will be used as the default for your FUTURE contributions. If you would like them to be applied to your previous contributions, be sure to select the relevant options in the pink boxes below. If you co-author ToL pages with other contributors, and individual authors have different license preferences, the first author's preference will determine the default license for a given page. You can also override a default license by selecting another license in the media or page data forms of individual contributions. For more detailed information about available licensing options, see ToL Use of Contributions.

Text Contributions

I permit the Tree of Life Web Project to use my TEXT contributions (web pages) in the following ways:

  • Text released under a creative commons licenses can be copied, reused, and distributed by the general public as long as attribution regarding the source of the information is maintained. Licenses are available that restrict the commercial use and the adaptation and modification of text. Copyright notices for materials released under this option are marked with the Creative Commons icon and contain a link to the deed for the selected license. For more information about these licenses, see the Creative Commons web site. We encourage ToL contributors to release their text contributions under a creative commons license, because these licenses will let us share materials with the Encyclopedia of Life (see ToL/EOL Collaboration). All creative commons licenses that permit the reuse and adaptation of text are compatible with the EOL intellectual property model, i. e., Attribution, Attribution-NonCommercial, Attribution-ShareAlike, and Attribution-NonCommercial-ShareAlike.

  • If you select this option, your text can be displayed only on the Tree of Life web site.

  • Materials in the public domain may be freely reproduced, distributed, modified, built upon, or otherwise exploited by anyone for any purpose, commercial or non-commercial, and in any way, with or without attribution to the original source.

  • Note that this will apply your default license only to previous contributions where you are the first author.

A Note About Changing Licenses for Previous Contributions

The ToL system allows you to change licenses for your contributions at any time with the touch of a button. However, you should be aware that eventhough you can change the license under which the ToL distributes your contributions, you cannot legally revoke a creative commons license. Once a copy of your work has been distributed under a given license, it will travel through cyberspace with that license attached. People will be able to use and redistribute your materials under the original license, even if you change the license for the ToL source file.
