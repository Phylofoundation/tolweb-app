Add media (images, sounds and movies)

  • To delete an image you must click on it first, then click the delete button .
  • To move an image you should delete it first and then re-insert it in your preferred location.
Insert images and media with the ToL Easy Editor
  1. Click on the place you want the image or media file to go in the text box.
    • Click to insert an image
    • Click to insert a sound, movie or document
  2. Search for an image or media file to insert. When you find what you want, click select to insert the image or media file into the text box. *For images you can click on "options" to select a size for your image. The default size is 250px high.
  3. To format images, first click on the image, then use the image formatting buttons (next to the insert image button).
  4. Click save and preview to see if the media file appears the way you want it to.
Insert images/media without the Easy Editor
  1. Click the "Add Media" button to search for media in the ToL database.
  2. Follow the instructions for working without the Easy Editor on the insert media page
