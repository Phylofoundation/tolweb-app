ToL Image : Size Options

Select a size* for the image you will insert into your page. For more information about size see below.

Choose a new size* (see below) for the selected image. This window will stay open, so you can try out different sizes. Click on Close Window when you are done.

Note that you may encounter difficulties when trying to resize very large images in Internet Explorer. If you cannot get the resizing to work, you need to delete the images and insert it again at a different size. Follow the options link in the image search results to do this.

Select a Size*
Size* ToL Image Code
 
  • *Size: this refers to the height of the image in pixels. The maximum size in which you can display an image is constrained by the size of the master image or by the maximum size allowed by the copyright owner. In order to fit on most people's screens, images should not be wider that 800 pixels or higher than 600 pixels. Most images should actually be quite a bit smaller than that, because you probably want to show your image along with some text or other images.
  • *Master: if the Master is selected, the image will displayed at the size of the master image or the maximum size allowed by the copyright owner.
