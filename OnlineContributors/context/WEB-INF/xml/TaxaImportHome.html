Taxa Import Workflow

Remove Clade
This interface allows you to removal all descendent taxa from a given basal node.
Taxa Import
This interface allows you to import taxa using the defined ingestion XML.
Object Management
This interface allows you to reattach objects attached to nodes in a given clade along with viewing various related logs. If you just want to move objects from a SINGLE node to another node, you can also use the Node Object Manager.
View Object Manifests
This interface allows you to view object manifests created for clades under construction.
View Taxa Import Records
This interface allows you to view taxa import records created during the construction of clades.
Object Manifest
This interface allows you to create a manifest (listing) of all objects (media, pages, etc) attached in a given clade. Note: only nodes with attachments will appear in an object manifest
Node Ancestors Correction
This interface allows you reset the ancestor relations for a node (adding them if they are not present).
Page Validity Check
This interface allows you to correct which page a given node believes it appears on.
Extinct Check
This interface allows you to cascade the extinct property throughout a clade rooted by the node-id passed.
