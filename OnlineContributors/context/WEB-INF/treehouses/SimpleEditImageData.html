What is copyright?

Copyright is a kind of law. It says that if you make something, then it belongs to you. If you write a book, or you write some music, or you paint a picture, then the thing you made belongs to you. Other people cannot make a copy of it unless you say that they can. Read more about copyright rules and issues.

I found this media file somewhere and have not received permission to use it.

why select this choice?

If you found this media file on the web or elswehere and did not receive permission in writing to use it, you cannot upload it to the ToL database until you receive permission from the copyright holder. Use this text in an email or letter to ask for permission. Note that there are some sites that may allow for use of media files on other web sites, however in the majority of cases the term "educational use" does not mean that you can post a media file on another site, but rather that you can download the file for use within a classroom.

Note also that if you can't get permission you may want to try creating your own media file by drawing or using a digital camera.

 

This media file is clearly marked as in the Public Domain.

why select this choice?

Select this choice if you are certain materials are considered to be in the public domain. When a media file is in the public domain the original copyright owner has released the work to the public, which means that you may use the media file in your treehouse page and that you can modify it to your liking. You must be able to provide the source of the media file and the name of the organization, and you should record who created the media file if that is available. Deciding whether a resource is in the public domain can be tricky, so unless a resource is clearly marked as public domain, you should not assume that it is freely available (e.g. just because a copyright owner is not listed does not mean a media file is in the public domain). Please view our list of web sites that provide media files in the public domain.
