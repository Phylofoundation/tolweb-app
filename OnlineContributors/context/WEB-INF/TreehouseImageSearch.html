Add an Image to a Treehouse Page

Find images to insert into your page

Images displayed on ToL pages are retrieved from the ToL image database. Use the fields below to find the image you want. Once you find an image you would like to use click "insert" to add the image to your page. Note that not all ToL images are available for use on all ToL pages. Your search will retrieve only images that have been released by their copyright owners for your intended use. See search tips for help.

Add a new media file to the ToL database

If you would like to use media that is not yet in the ToL database, you can upload it here. Please be sure to provide data for the media in the form that opens after the media is uploaded.

Choose a file to upload from your computer then click on the Upload New Media button.

Search tips

The following tips will help you find the maximum number of results and/or complete a successful search for media in the ToL database.

  • Less is more. Typing text into only one search field will help maximize your results. Also, typing in less text will help avoid typos and misspellings.
  • Don't type something into each search field. The exception is if you are looking for a very specific type of media file.
  • Don't use spaces between words that you are searching for. The exception is if you are searching on a scientific name for a group of organisms or the name of a copyright holder. However, we still recommend you initially type in only one word.
  • If you contributed the media yourself and you are the copyright owner, you can search by your name in the Copyright Owner field.
  • If you know the name of the file you contributed you can use the File Name field.
  • When looking for media you did not contribute, type search criteria in the "Attached to Group" and "Text in Any Media Data Field" sections. We recommend that you try typing in what you are looking for in "Attached to Group" first, because if we have an image or other media file of this organism you should be able to find it in this way. However, if you are using a common name for a group of organisms (such as butterfly rather than Lepidoptera) you may be more successful if you type text into "Text in any Media Data Field".
  • See help with scientific names to find out some of the scientific names you can use to search for media of various organisms.
  • For additional examples see help finding media in the ToL database
