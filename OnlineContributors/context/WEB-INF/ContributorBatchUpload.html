Batch Uploading ToL People

Below, you can upload batches of people to the ToL database.

  1. Create an XML file containing the data (name, email, permissions, etc.) for each of the image files. See the instructions for Creating People Batch Data XML Files below
  2. Use the Browse (Choose File) button to locate the XML file
  3. Check the following box to update people information that are already in the database. (This happens when the xml file contains an email address that matches an existing record in the database.):
  4. Click on the Upload File button to initiate the upload.
  5. Once the upload is complete, you will see a confirmation message in this window.
    • If the upload was successful, you'll get an overview of the people that were uploaded. If the results of the upload were not what you expected, you can delete all of the newly uploaded people from the overview page.
    • If the upload fails, please try again. If it won't work the second time, please contact and we will investigate the problem.
