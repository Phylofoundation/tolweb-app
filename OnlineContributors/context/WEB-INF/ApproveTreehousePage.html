Submit Treehouse '' to the ToL for Publication

Check off the areas below before the treehouse is submitted for publication. It is especially important that you ensure personal information and copyright rules are followed.

Comments that you add will be published in the Page Communication/History section (step #7) of the student's treehouse, and they will also be sent via email to you and the learning materials editor.

Please check that the areas listed below have been reviewed for and meet ToL guidelines.

  • Student Personal Information
    • Make sure that student personal information (e.g. name, photos, email, address) is not displayed unless the student has permission in writing from the parent/guardian and school.
  • Copyright of Media Files (Review Copyright Rules and Issues | The Copyright Quiz)
    • Make sure that all images have copyright and/or credits information listed underneath them and all media files have copyright/credits information when they are clicked on.
    • Make sure that images marked as copyrighted to the student were created by the student (the default copyright owner when an image/media file is uploaded is the person logged in, and this can confuse students).
    • Make sure that the source for all images found outside of the ToL on the web is indicated, e.g. the name of the site and its URL (web address), and that permission has been received in writing for any copyrighted media files found on the web.
  • Content/Style/Spelling/Grammar (Review ToL Design Guidelines and the Basic Treehouse Guidelines)
    • Please approve student work unless there are glaring spelling and grammar errors or lack of basic content. It is easy for us to reformat student pages and we realize that there are quirks in the Easy Editor that may cause formatting problems for students.
